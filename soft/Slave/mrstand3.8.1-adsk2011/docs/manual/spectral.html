Spectral Rendering

Spectral rendering is supported in mental ray. It is enabled by using the special colorprofile "micsSpectrum" for the render color space. mental ray spectra contain wavelengths between 400 and 700 nano-meter. This domain is split into 31 equidistant intervals of 10 nano-meter each. Spectral rendering requires shaders specifically written for spectral rendering. It is not possible to mix shaders returning colors with shaders using spectra. Therefore, spectral rendering requires that all involved shaders understand spectra. Spectral rendering has the following restrictions:

  • scanline rendering and the rasterizer are not supported;
  • final gather mode is not supported;
  • unlike spectral caustic photons, spectral globillum photons are not stored with their full spectrum, but only with RGB tristimulus values.
