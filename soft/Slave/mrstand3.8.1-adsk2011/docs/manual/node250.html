Image Information: imf_info

The image information utility imf_info is started as

   imf_info [options] file...

It will display information about all named image files, including resolution, number of color components, number of bits per component, gamma factor if available, top-down or bottom-up line ordering, data type, and the image format. There is only one option:

-v

Verbose output prints messages showing what imf_info is doing, and a version banner.

-p

3.1 Plot a simple pixel intensity histogram.

-m

Prints information about sample pass files.

-l

Prints pyramid layer resolutions for memory-mapped texture files.

-L

Prints more detailed pyramid layer information, including deviations from the expected resolutions. This is useful for map files created with imf_copy -c.
