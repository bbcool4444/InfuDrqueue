Polygon Vectors

Element type: miSCENE_GEOVECTOR
Data type: miGeoVector[ ]
Sizes: int no_vectors
Defaults: all nulls

The vector list is a miGeoVector array. It is partitioned into sections as described by the vect_info structure in miPolygon.
