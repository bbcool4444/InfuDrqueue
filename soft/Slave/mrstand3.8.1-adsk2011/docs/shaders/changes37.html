Changes in version 3.7

The following changes were made in version 3.7 of the shaders. Please note that this list may be incomplete:

  • Added one new parameter to fg_mode to mib_lightmap_write.
  • Added new variant misss_fast_shader_x of existing subsurface shader which provides result values as separated components instead of the finally composited color value.
