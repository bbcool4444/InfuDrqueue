Changes in version 3.6

The following changes were made in version 3.6 of the shaders. Please note that this list may be incomplete:

  • Added light mode input parameter on physics shaders, and support for instance light lists.
  • Added light mode input parameter on physical subsurface shader, and support for instance light lists. Added support for instance light lists in fast subsurface shaders.
